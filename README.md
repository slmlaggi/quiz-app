# This is a quiz application written in python intended to be run as a text-based app in the terminal

Parts of the code, mainly the sqlite parts, are written by GPT-4 and refactored manually.

Add your own quizzes as .txt files in the ./quizzes directory. Format should be as follows:
Line 1: Quiz Title
Line 2: Question 1
Lines 3-6: 4 multiple-choice answers
Line 7: Answer (A, B, C, or D) of the question

More questions shall be added following the instructions of lines 2-7. Ensure there are no lines between questions and each question must occupy 6 lines as formatted.
